# Student CGPA Prediction with PySpark

## Project Overview
This project uses PySpark and Machine Learning to predict students' Final CGPA based on:
- Attendance Percentage
- Study Hours Per Day
- Previous CGPA

## Technologies Used
- Python
- PySpark
- Spark MLlib

## Machine Learning Model
- Linear Regression

## Evaluation Metrics
- R2 Score
- RMSE

## Project Structure

data/
src/
README.md
requirements.txt

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the project:
python src/pyspark_project.py
